# Alfred-Workflow
Alfred Workflow for developer

## Requirements
* [Alfred App](https://www.alfredapp.com)
* [Alfred Powerpack](https://www.alfredapp.com/powerpack/)

## [Download](https://github.com/asktao/Alfred-Workflow/releases/latest)

## Commands
* t 
* t {query}

* md5 {query}

* urlencode {query}
* urldecode {query}

* base64_encode {query}
* base64_decode {query}
